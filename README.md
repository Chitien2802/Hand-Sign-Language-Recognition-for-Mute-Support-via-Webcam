# Hand-Sign-Language-Recognition-for-Mute-Support-via-Webcam
Real-time hand sign language recognition using deep learning. Captures hand gestures via webcam and converts to text/speech for mute support. Features CNN-based detection, live webcam integration, accessible interface. Tech: Python, TensorFlow/Keras, OpenCV, NumPy, Scikit-learn. Assistive technology for inclusive communication.
